# Mirha Rainbow Interiors

A beautiful handmade art and decor website built with Next.js 14 and Tailwind CSS.
Products are managed via Google Sheets - no coding needed for the owner!

## Quick Start
npm install

Edit .env.local with your credentials
npm run dev

Open http://localhost:3000
  
## Tech Stack
- Framework:  Next.js 14 (App Router)
- Styling:    Tailwind CSS + Framer Motion
- CMS:        Google Sheets
- Emails:     Nodemailer (Gmail)
- Hosting:    Vercel (Free tier)
- Images:     Google Drive or Cloudinary

## Cost
- Domain (.com): approx PKR 3,000 per year
- Everything else: FREE

## Managing Products
Edit the Google Sheet to add, update or hide products.
The website refreshes automatically every 10 minutes.

See SETUP_GUIDE.md for full instructions.
